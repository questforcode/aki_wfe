Metadata - Block Elements

Metadata

The metadata we have learned to use are: meta, title, link, script, and style. They are all explained inside the comments of the head.

Block Elements

Section

The elements that are reflected in the web page's outline, just like the chapter titles of a book.

Before HTML5, these elements were the body and headings. This was an implicit sectioning. However, in HTML five, we have new semantic elements. These explicit sections are: section, header, footer, nav, article and aside. We can organize as we want.

  • section: very specific section defined by the user;
  • header: exactly what the name says, there can be as many as needed but only one per section;
  • footer: exactly what the name says, there can be as many as needed but only one per section;
  • nav: a special section that allows us to establish navigational elements such as menus;
  • article: sections you consider interesting independently, like blogs;
  • aside: side information, newsletter stuff, extras etc.

Grouping

The elements that are not reflected in the web page's outline, like the contents of a chapter in a book.
