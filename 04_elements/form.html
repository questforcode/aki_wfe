Forms

The following controls are available when it comes to making forms:

  • input - text : a box for inputing text, like usernames, passwords, etc;
  • textarea: bigger text input areas;
  • checkbox: boxes to check options (boolean so true or false);
  • radiobutton: multiple option check boxes;
  • select / options: dropdown list
  • buttons: send buttons, stuff like that, etc. sends everything;
